# 6CCS3PRJ
KCL 6CCS3PRJ - AI opponent for Ulti card game

## CLI

- `npm run build` to compile project
- `npm start` to run simulation
- `npm start -- [args]` to pass arguments
- `npm start -- -h` to see CLI help
- `npm test` to run TS module tests
